============
Installation
============


Stable release
--------------

To install ARAS control service protocol, run this command in your terminal:

.. code-block:: console

    $ pip install aras_control_service_protocol

This is the preferred method to install ARAS control service protocol, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for ARAS control service protocol can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/cartovarc/aras_control_service_protocol

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/cartovarc/aras_control_service_protocol/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/cartovarc/aras_control_service_protocol
.. _tarball: https://github.com/cartovarc/aras_control_service_protocol/tarball/master
